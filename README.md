# Airline Customer Satisfaction

Проект посвящён анализу и прогнозированию удовлетворённости клиентов авиакомпании.

## Данные
Используется открытый датасет с информацией о клиентах:
- демография
- тип перелёта
- оценки сервиса
- задержки рейсов

Целевая переменная: `satisfaction` (0 — не удовлетворён, 1 — удовлетворён).

## Что сделано
- EDA и очистка данных
- Обработка выбросов и пропусков
- Кодирование категориальных признаков
- Обучение моделей:
  - Logistic Regression
  - Random Forest
- Оценка качества по ROC-AUC
- Анализ важности признаков
- Подготовка production-ready pipeline
- Сохранение модели (`joblib`)

## Стек
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- scikit-learn

## Результат
Random Forest показал улучшение качества по сравнению с логистической регрессией.
